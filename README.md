## データベース関連


### DB構造

treeとtaskは生成、それを元にサーバー側で各contentを作成
<br> →　プロンプト削減のため
<br>主に表示するのはtreeと各タイトルのみ
<br>content作成はtitle,children,statusを作る
<br>UI上で各タスクに触れた時にdescription==nullだったら説明作成&DBのdescription更新とかにすればプロンプト減らせる...かな
